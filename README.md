**WCAG Color Contrast Checker** - Offline web app for web devs &amp; designers.

## What it does
- Enter fg/bg colors, font size/weight
- Instant WCAG AA/AAA pass/fail
- No tracking, runs in browser

## Who it&#39;s for
Web developers, designers, agencies avoiding ADA lawsuits (~$10k avg settlement) and meeting client/SEO standards.

## Quick Start
1. Download/clone repo
2. Open `index.html` in any browser
3. Check contrasts

Live demo: https://harryherzberg.github.io/contrastcheck (TBD)
